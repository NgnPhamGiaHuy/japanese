Act as a senior engineer refactoring code for scalability.

## OBJECTIVE

Improve code quality WITHOUT changing behavior.

---

## RULES

- Split large files (>150–200 lines)
- Extract logic into hooks
- Move API calls into services
- Remove duplication

---

- Over-engineer

---

## OUTPUT

- Clean, modular code
- Feature-based structure
- Small, readable files
